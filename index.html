Frequently Asked Questions

How do I get a Google Gemini API key? expand_more

Step-by-Step Guide:

  1. Visit Google AI Studio:
    Go to aistudio.google.com
  2. Sign in:
    Use your Google account to sign in
  3. Get API key:
    Click on "Get API key" in the left sidebar
  4. Create new key:
    Click "Create API key" button
  5. Select project:
    Choose "Create API key in new project" (recommended)
  6. Copy your key:
    Copy the generated API key and save it securely
  7. Add to extension:
    Paste the API key in MoodleGPT settings
üí° Important: The API key is free to use with generous limits. Keep your key private and never share it publicly.
Do I need a Gemini API key? expand_more
