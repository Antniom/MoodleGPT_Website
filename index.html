Question: What is the time complexity of binary search?
High Confidence: 99%

Answer: O(log n)

📚 Detailed Explanation:

Binary search has a time complexity of O(log n) because:

  • Each comparison eliminates half of the remaining elements
  • The search space is divided by 2 at each step
  • Maximum comparisons needed = log₂(n)

Install from Chrome Web Store Setup Guide

Quick Setup Guide

Get MoodleGPT Pro running in minutes

Step 1: Install from Chrome Web Store

Click "Install from Chrome Web Store" above. The extension will be automatically installed to your browser. No need to download files or enable developer mode!

Step 2: Get Your Free Gemini API Key

Visit Google AI Studio, sign in with your Google account, and generate a free API key. The key is yours to keep and use.

Step 3: Configure the Extension

Click the MoodleGPT Pro icon in your browser toolbar, paste your API key in the input field, and click "Save". Your key is stored securely in your browser.

Step 4: Start Using on Moodle

Navigate to any Moodle quiz, click the extension icon to open the popup, select your preferred mode (Study/Exam/Hint), and use hotkeys Alt+S to extract answers!
