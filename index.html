Installation Guide

Installation Guide

Step-by-step instructions to get MoodleGPT Pro running

Step 1: Download the Extension

Download the MoodleGPT Pro extension files from the link above. Extract the ZIP file to a folder on your computer.

Step 2: Enable Developer Mode

Open Chrome/Edge, go to Extensions (chrome://extensions/), and enable "Developer mode" in the top right corner.

Step 3: Load Unpacked Extension

Click "Load unpacked" and select the NewApproach folder from the extracted files.

Step 4: Configure API Key

Get a free Gemini API key from Google AI Studio, then add it in the extension settings.
